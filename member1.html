Smart Traffic Management System

A DAA-based smart city traffic system designed to optimize signal control, emergency routing, congestion handling and public transport prioritization.

Overview Scenarios Analytics Summary

System Overview

The traffic network of the city is modeled using graphs, where intersections represent nodes and roads represent edges. Vehicle movement is handled using queues, while routing decisions are taken using classical shortest-path algorithms from DAA.
