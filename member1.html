Adaptive Signal Control

Each lane is treated as a queue. Green time is assigned using a greedy strategy.

↓
Store count in array
↓
Calculate total vehicles
↓
Assign green time proportionally
↓
Update signals

Emergency Vehicle Routing

Ambulances get the shortest path with signal priority. 

Emergency detected
↓
Push into Priority Queue
↓
Run Dijkstra on road graph
↓
Override traffic signals
↓
Clear path
Algorithms:
  • Graph
  • Dijkstra
  • Priority Queue

Summary

  • Queues manage vehicle waiting
  • Graphs represent road network
  • Dijkstra ensures shortest routes
  • Priority Queue saves lives
  • Charts help decision-making
Time Complexities:
  • Dijkstra → O(E log V)
  • BFS → O(V+E)
  • Priority Queue → O(log n)
